# Mohamed-Salah-vs-Kylian-Mbapp-An-Attacking-Performance-Analysis-2017-18-2024-25-
This project analyzes Mohamed Salah and Kylian Mbappé's attacking performance from 2017/18–2024/25 using SQL and Google BigQuery. The goal was to determine which player produced the more efficient and consistent attacking contribution using advanced per-90 metrics.
